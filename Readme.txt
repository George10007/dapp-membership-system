(a)環境: 操作系統: Windows 10或以上版本
         區塊鏈網路: 以太坊Sepolia測試網
         區塊鏈節點: MetaMask預設節點

(b)軟體: VScode
	 Node.js
	 JavaScript函式庫:React.js, Ethers.js

	 MetaMask
(c)Dataset: 群組成員資料，儲存於區塊鏈上

(d)實作流程:
	 1. 先行安裝vsCode(編譯器), Node.js  版本v20.16.0(用nvm安裝，指定版本20.16.0)(JavaScript開發環境)以及瀏覽器延伸功能下載Metamask(錢包)
	 2. 在vsCode中打開專案資料夾(dapp-membership-system)
	 3. 在vsCode中用terminal(終端)分別到membership-system和backend的目錄下(輸入)執行 npm install 指令，安裝相關依賴
	 4. 在瀏覽器的延伸功能找到Metamask，並依指示建立Metamask錢包，接著選測試網路中的sepolia，用來和以太坊連接
	 5. 在sepolia水龍頭領取sepolia測試幣(錢包內需有以太幣)(水龍頭網址https://www.alchemy.com/faucets/ethereum-sepolia)
	 6. 在vsCode中trerminal在membership-system的目錄下輸入 npm start 執行前端(網頁介面)
	 7. localhost:3000 網頁會執行系統介面，可以看到儲存的資料，需有測試幣才可以進行操作
